# homebrew-vllm-mlx-ui

Homebrew tap for [vllm-mlx-ui](https://github.com/clickbrain/vllm-mlx-ui) — Apple Silicon LLM inference server with browser-based dashboard.

## Install

```bash
brew tap clickbrain/vllm-mlx-ui
brew install vllm-mlx-ui
```

## Upgrade

```bash
brew update && brew upgrade vllm-mlx-ui
```
